### 梦网 --- montnets

配置类:`org.dromara.sms4j.montnets.config.MontnetsConfig`

| 字段名称              | 作用      | 默认值                         |
|-------------------|---------|-----------------------------|
| `accessKeyId`     | 账号      | `null`                      |
| `accessKeySecret` | 密码      | `null`                      |
| `url`             | 请求地址    | `http://ip:port`            |
| `api`             | 接口名称    | `/sms/v2/std/send_template` |
| `templateId`      | 默认模板id  | `null`                      |
| `templateParam`   | 默认模板参数名 | `null`                      |
